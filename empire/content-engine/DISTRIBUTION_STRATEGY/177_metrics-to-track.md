#### Metrics to Track

**Engagement Metrics**:

- View count
- Completion rate
- Like rate
- Comment rate
- Share rate
- Save rate

**Discovery Metrics**:

- Impressions
- Reach
- Profile visits
- Follow conversion rate
- Click-through rate (links)

**Quality Metrics**:

- Average watch time
- Re-watch rate
- Audience retention graph
- Traffic sources
- Demographics reached
