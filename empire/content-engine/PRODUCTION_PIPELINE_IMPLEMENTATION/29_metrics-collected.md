### Metrics Collected

The pipeline automatically tracks:

1. **Performance Metrics:**
   - Total duration per episode
   - Duration per step
   - API response times

2. **Cost Metrics:**
   - Cost per API service
   - Cost per episode
   - Running total

3. **API Metrics:**
   - Total API calls per service
   - Success/failure rates
   - Retry counts

4. **File Metrics:**
   - Audio files generated
   - Images generated
   - Videos generated

5. **Error Tracking:**
   - Error messages
   - Stack traces
   - Context (which step failed)
