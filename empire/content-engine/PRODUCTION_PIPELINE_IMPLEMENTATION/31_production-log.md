### Production Log

All production events are logged to:

```
output/production-log.json
```

Sample log entry:

```json
{
  "episodeNumber": 1,
  "status": "completed",
  "duration": "15.5 minutes",
  "timestamp": "2025-10-07T12:15:30.000Z",
  "error": null,
  "cost": "$1.82",
  "apiCalls": 19
}
```
